# Projeto: Calculadora Simples em Python

## Descrição
Este programa é uma calculadora simples que realiza operações básicas: adição, subtração, multiplicação e divisão.  
O programa valida as entradas do usuário e trata casos de divisão por zero ou entradas inválidas.

## Funcionalidades
- Solicita dois números ao usuário.
- Permite escolher a operação: +, -, *, /.
- Valida entradas para aceitar apenas números.
- Trata divisão por zero.
- Estruturado em função para melhor organização.

## Tecnologias utilizadas
- Python 3

## Como executar
1. Clone este repositório ou baixe o arquivo `calculadora.py`.
2. Abra o terminal (ou prompt de comando).
3. Execute o programa com:
   ```bash
   python calculadora.py
